# Research Notes

## Event

CrowdStrike Global Outage

July 19, 2024

## Background

CrowdStrike released a Falcon sensor update that contained a faulty configuration file.

The update caused Windows computers around the world to crash and display the Blue Screen of Death (BSOD).

## Key Facts

- Faulty Falcon update
- Millions of Windows devices crashed
- Airlines, banks, and hospitals affected
- Recovery required manual intervention

## Software Development Lessons

- Test updates before release
- Review code carefully
- Use staged deployment
- Monitor systems after deployment
